## Chat React App

This is Chat React App made with React. 

### Description

The project is a chat react app made with React.js, Sockets, Rest APIs, and Hooks.  

### Getting Started

#### Dependencies

- React Icons: 4.3.1
- AntDesign Icons
- Axios
- React Chat Engine

#### Installing

- npm i react-icons
- npm i @ant-design/icons 
- npm i axios
- npm i react-chat-engine

#### Executing app

- npm start 

#### Version Control

- React: 17.0.2
